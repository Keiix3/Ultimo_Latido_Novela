# Último Latido - Novela Interactiva

## Descripción
Novela interactiva de terror y supervivencia ambientada en Buenos Aires durante un apocalipsis zombie. El jugador toma decisiones que afectan el destino de los personajes y el desarrollo de la historia.

## Características
- Motor de novela propio en Python.
- Historia ramificada con múltiples finales.
- Sistema de estado oculto (variables que afectan los diálogos).
- Interfaz en consola con colores y estilos usando la librería `rich`.

## Requisitos
- Python 3.8 o superior.
- Entorno virtual (venv) recomendado.

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar el juego
python main.py
